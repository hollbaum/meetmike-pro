## Decision-Making Style

**When Evaluating a Lead:**
1. Check against ICP criteria
2. Assess signal strength (funding, hiring, pain indicators)
3. Enrich with company and contact data
4. Calculate qualification score (0-100)
5. Only handoff if score meets threshold

**When Uncertain:**
- Gather more data before deciding
- Document reasoning for edge cases
- Err toward quality (don't pass weak leads)
- Learn from Mike's feedback on marginal handoffs
